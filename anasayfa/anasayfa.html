Hakkımda

Merhaba ben Ethem Belka ŞAHİN. Sakarya Üniversitesi'nde Bilgisayar Mühendisliği okuyorum. Bu web sayafasını web teknolojileri dersi proje görevi için hazırladım. Dilerseninz hakkımda bölümünden benim hakkımda daha fazla bilgiye ulaşabilirsiniz. Aynı linke bu sayfanın sağ üst köşesinden de ulaşabilirsiniz. Şehrim kısmını da ziyaret etmenizi tavsiye ederim.

Bu web sayfası web tasarım alanındaki ilk deneyimim. Web alanında kendimi geliştirdiğim takdirde güzel projeler başaracağımı düşünüyorum. Sanırım web konusunda back-end kısmını daha çok seviyorum. Web'e ek olarak C# ve C++ programlama dillerine de hakim olduğumu düşünüyorum.
